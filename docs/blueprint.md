# **App Name**: HariumAI

## Core Features:

- Text Conversation: Conversational Text Generation using the Gemini API to generate natural language responses based on user input.
- Image Generation: Image Generation using the Gemini API to create images based on text prompts; tool that attempts to incorporate all aspects of user text prompts, especially emotion
- Speech Output: Text-to-Speech output to convert generated text into spoken words for accessibility and hands-free interaction.
- Interaction Handling: Multi-modal Interaction enabling users to seamlessly switch between text, images, and speech.

## Style Guidelines:

- Primary color: White (#FFFFFF) to evoke a sense of clarity and simplicity.
- Background color: White (#FFFFFF), for high readability and a clean interface on web.
- Accent color: Black (#000000) for text and essential UI elements to ensure clarity and contrast.
- Body and headline font: 'Inter' (sans-serif) for a modern, clean, and readable design.
- Use clean, minimalist icons in black to maintain a consistent and modern aesthetic.
- Implement a clean and intuitive layout with a focus on ease of use and accessibility, ensuring a seamless user experience across different devices.
- Incorporate subtle animations for feedback and transitions, enhancing the overall user experience without being intrusive.
